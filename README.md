# File-Based Blog System

Current Functionalities: 

***Post Creation***
* Custom slug or generate it from the title.
* Support markdown for content editing with images using EasyMDE.
* Cover image is independent from the images within the text and resizing it.
* Support several files uploading.
* Selection of multiple tags, categories.
* Support scheduling the post or making it as a draft, ad validating the scheduling time.
* Validating the inputs and handling duplicate if there is a repeated slug.

***Homepage***
* Displays all the posts accroding to the date of creation.
* Filtering according to the tags and categories.
* Searching across the post title, description and body content.
* Displaying the reading time, number of likes, comments and attachments of each post, and the author of the post.
* Working on the page which displays each post individually still not complete).
* Pagination for the posts 2 per page.

*** Next***
* Refactoring parts of the code.
* working on the page which displays each post individually.
* Login page is done only frontend still haven't started the backend of the login.
* Admin panel to include creation of tags, categories, posts and users and managing them through editing and deleting.
